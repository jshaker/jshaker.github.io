Ovrpass

Talent Pool Manager

Ovrpass Official Website

Overview

Ovrpass is an easy to use drop-in SaaS recruitment chatbot. Ovrpass gives you the ability to define a custom flow of questions for screening and hiring candidates through the Ovrpass dashboard. Once you’re satisfied with the bot configuration, adding it to your website is a breeze as it’s just a matter of adding one line of code to your website.

Contributions

I built Ovrpass from the ground up using React, Redux, Firebase and Stripe. The chatbot widget is a lightweight bundle written in React with the focus on keeping it thin as to avoid slowing down the customers’ pages. The chatbot flow is essentially a state-machine which leverages Firebase’s realtime functionality to maintain an updated state on both client and server. The dashboard was built using React and Redux as the logic tends to be more complex than the widget. Stripe was used to manage users’ subscriptions as this is a SaaS product.
